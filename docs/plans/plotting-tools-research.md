# Plotting Tools Research Summary

Research conducted March 2026. Analysis of the most popular and well-regarded plotting/outlining tools for fiction writers, covering features, workflows, AI integration, and differentiators.

---

## 1. Popular Plotting Tools

### Tier 1: Dedicated Plotting Tools

**Plottr** — The market leader for dedicated plotting software.
- Visual timeline as the core metaphor
- 40+ story structure templates (Hero's Journey, Save the Cat, Snowflake, 3-Act, Story Circle, 12-Chapter Mystery, Romance, etc.)
- Scene cards organized on plotlines with drag-and-drop
- Color-coded plotlines for visual differentiation of storylines/subplots
- Character sheets, location tracking, notes system with tagging
- Series timeline view for multi-book projects
- Export to Word and Scrivener; import from Scrivener and Snowflake Pro
- Pro tier adds cloud sync, real-time collaboration, browser access
- Pricing: $9.99/mo base, $14.99/mo Pro, or $150/$599 lifetime

**Campfire Writing** — The most modular and world-building-oriented tool.
- 18+ modules sold individually (Characters, Timeline, Maps, Calendar, Arcs, Relationships, etc.)
- Timeline module: customizable canvas with card panels, section dividers, and labels
- Calendar module: fictional calendars with custom eras, months, moons, seasons
- 10 pre-built story structure templates (3-Act, 7-Point, 27-Chapter, Fichtean Curve, Heroine's Journey, Hero's Journey, Romancing the Beat, Save the Cat, Story Circle, Story Grid)
- Gantt chart view for visualizing narrative timespans at macro/micro levels
- Arcs module and Relationships module (web visualization)
- Interactive maps for worldbuilding
- Modular pricing from $0.50–$2.00/mo per module; ~$12/mo for everything

**NovelCrafter** — The AI-first plotting tool.
- Grid view: drag-and-drop acts, chapters, scenes
- Matrix view: spreadsheet-style full-structure view
- Outline view: streamlined single-page structure
- The Codex: central wiki for characters, locations, lore — shared across series
- Relations: build family trees, org charts, connection networks
- Mention Timeline: visualize when characters/elements appear
- Appearance Heatmap: discover hidden character connections
- Characters per Scene: identify overcrowded or sparse scenes
- Scene Labels for tracking draft stages
- AI integration with 300+ models (OpenAI, Anthropic, Google, Meta, Mistral, local models)
- Scene Beats: tell the AI what should happen next for directed story development
- Workshop (AI chat): brainstorm, get feedback on scenes
- Starting from $4/mo, 157,000+ authors

### Tier 2: Writing Suites with Strong Plotting Features

**Scrivener** — The veteran writing suite with built-in plotting.
- Corkboard: virtual index cards with synopses; moving cards rearranges manuscript
- Outliner: customizable columns with synopses, word counts, metadata
- Binder: hierarchical ring-binder organization (parts > chapters > scenes)
- Scrivenings mode: edit sections individually or as a single document
- Metadata system: color-coded labels, status tracking, custom fields
- Collections: dynamic document groups for tracking viewpoints, flashbacks, revision stages
- Split-view editing: up to 4 documents simultaneously
- Custom document templates for character sheets, location profiles
- One-time purchase, Mac/Windows/iOS

**Dabble Writer** — Cloud-based writing suite with Plot Grid.
- Plot Grid: organize with plotlines, plot points, and plot labels
- Character profiles with property lists, banners, avatars
- Worldbuilding tools and Story Notes
- Focus mode for distraction-free writing
- Cloud-based with auto-sync

### Tier 3: Lightweight/Specialized Tools

**Wavemaker** — Progressive web app with multiple planning paradigms.
- Snowflake Method tool: start small, progressively add detail via expanding cards
- Cards Planning Board: kanban-style scene/idea organization
- Timeline tool: chronological event placement
- Grid Planner: inspired by JK Rowling's planning grid
- Research database with hashtag organization
- Offline-capable PWA, works on any device
- Free/affordable

**PlotFactory** — Web-based plotting with character-centric views.
- Plot Helper Outlines with pre-made story templates
- Plot Organizer: plot points grouped into acts
- Chapter/scene divisions with character viewpoint identification
- Drag-and-drop scene reorganization
- Universe Notebook for tracking fictional worlds across books
- Character lists with avatars
- Writing statistics including character attention metrics

**Shaxpir** — Writing suite with emotional arc tracking.
- Manuscript Builder with drag-and-drop chapter reordering
- World-Building Notebook linked to manuscript chapters
- Emotional Journey: sentiment analysis that automatically plots emotional peaks/valleys
- Unique automated emotional arc visualization

## 3. What Makes Great Plotting Tools Stand Out

### The Differentiators

**1. The Visual Timeline as Primary Metaphor (Plottr's strength)**
The single most valued feature across reviews. Writers want to SEE their story structure — parallel horizontal swimlanes where each storyline is a row, and scenes/beats are cards placed along the timeline. Moving a card moves the scene. This provides an instant spatial understanding of pacing, subplot weaving, and structural balance that text-based outlines cannot match.

**2. Structure Templates That Teach (Plottr, Campfire)**
Great tools don't just organize — they teach craft. Providing 10–40+ story structure templates (Hero's Journey, Save the Cat, 3-Act, Story Circle, etc.) that auto-populate the timeline with beats gives writers both a framework AND a learning tool. The best implementations let writers customize templates and create their own.

**3. The Codex/Story Bible (NovelCrafter, Campfire)**
A central, searchable wiki of all story elements (characters, locations, lore, items) that stays consistent across scenes and books. The key differentiator: elements are *linked* — clicking a character in a scene card takes you to their profile; the profile shows everywhere they appear. NovelCrafter's Codex shares across an entire series.

**4. Multiple View Paradigms (NovelCrafter)**
Offering the same data through multiple lenses: timeline view, grid/matrix view, outline view, and single-scene view. Writers switch between these depending on whether they're doing macro-structural work or micro-scene work.

**5. AI as Brainstorming Partner, Not Replacement (NovelCrafter)**
AI that helps writers think through plot problems without taking over. Scene beats where the writer tells the AI what should happen and gets suggestions. Chat-based brainstorming. AI-powered context from the Codex so the AI knows the story world.

**6. Modular Depth (Campfire)**
Letting writers use only the modules they need. Not every writer needs interactive maps or custom calendars, but fantasy writers might need both. Pay-for-what-you-use avoids overwhelming simpler projects.

**7. Frictionless Connection Between Planning and Writing**
The best tools connect the outline to the manuscript. Scrivener does this natively (corkboard cards ARE the manuscript sections). Plottr exports to Scrivener. NovelCrafter integrates writing and planning.

### What Makes Tools Mediocre

- **Text-only outlines** without visual spatial representation
- **No story structure guidance** — just blank containers
- **Siloed data** — characters not linked to scenes, locations not linked to chapters
- **Rigid structure** — forcing a specific organizational paradigm instead of letting writers choose
- **No export path** — outline data trapped in the tool with no way to bring it to a writing environment
- **Feature overload without progressive disclosure** — showing everything at once instead of letting writers add complexity as needed

## 4. Common Plotting Workflows

### Workflow A: Template-First (Most Popular for New Writers)
1. Pick a story structure template (Save the Cat, 3-Act, Hero's Journey)
2. Template auto-populates timeline with labeled beats/milestones
3. Fill in each beat with scene ideas — short synopses on cards
4. Add character arcs as parallel plotlines
5. Add subplot plotlines
6. Color-code by storyline, POV character, or act
7. Rearrange scenes until pacing feels right
8. Export outline to writing tool and begin drafting

### Workflow B: Bottom-Up / Discovery (Popular for Pantsers Who Want Structure)
1. Brainstorm scene ideas as individual cards without order
2. Use a kanban/card board to group related scenes
3. Drag cards into rough chronological or narrative order
4. Identify gaps and add connecting scenes
5. Apply a structure template retroactively to check pacing
6. Assign scenes to chapters

### Workflow C: Character-Driven
1. Create detailed character profiles with goals, flaws, and desired transformations
2. Map character arc beats (First Impression > Goal > Flaw > Aha Moment > Action > Transformation)
3. Build plot events that force characters to confront their flaws
4. Layer multiple character arcs as parallel plotlines
5. Ensure character arcs intersect and influence each other
6. Check for scenes where characters are absent too long (heatmap/presence tracking)

### Workflow D: World-First (Fantasy/Sci-Fi)
1. Build world encyclopedia: locations, magic systems, cultures, history
2. Create fictional calendar with custom time systems
3. Place historical events on timeline
4. Build character profiles connected to world elements
5. Plot story against world timeline
6. Use maps to track character movement and geography

### Workflow E: Series Planning
1. Create series-level timeline with overarching plotlines
2. Break into individual book timelines
3. Share a story bible/codex across all books
4. Track which characters appear in which books
5. Ensure continuity across installments
6. Plan foreshadowing and payoff across books

## 5. AI Integration in Modern Plotting Tools

### Current State of AI in Plotting Tools

**NovelCrafter** is the clear leader in AI-integrated plotting:
- **Scene Beats**: Writer describes what should happen; AI suggests how to execute it
- **Workshop Chat**: Conversational brainstorming about plot problems, character motivation, pacing
- **Codex-Aware AI**: AI reads the story bible so suggestions are consistent with established world/characters
- **Multi-Model Support**: OpenAI, Anthropic, Google, Meta, Mistral, OpenRouter (300+ models), plus local models via LM Studio/Ollama
- **BYOK Model**: Users bring their own API keys — no markup on AI costs
- **Custom Prompts**: Tailor AI behavior with variables and personalized shortcuts

**Plottr** has minimal AI integration:
- Education Vault with "AI Search" in the Pro + Community tier
- No direct AI-assisted plotting or brainstorming in the core product
- The plotting workflow remains manual/template-driven

**Campfire** has no significant AI integration:
- Focuses on manual tools and modules
- No AI brainstorming or scene generation

### Key AI Plotting Use Cases

1. **Brainstorming**: "What if" scenario exploration — asking AI to suggest plot complications, twists, or alternative directions
2. **Beat Generation**: Given a story structure, AI suggests specific scene ideas for each beat
3. **Consistency Checking**: AI reads the codex/bible and flags potential continuity issues
4. **Character Voice**: AI helps develop distinct character voices and dialogue patterns
5. **Plot Hole Detection**: AI analyzes the outline for logical inconsistencies or unresolved threads
6. **Scene Expansion**: Writer provides a beat/synopsis; AI expands into a more detailed scene plan
7. **Pacing Analysis**: AI evaluates the rhythm of action, tension, and rest across the timeline

### AI Integration Best Practices (from Tool Analysis)

- **AI should augment, not replace** — writers still drive all creative decisions
- **Context-awareness is critical** — AI must know the story bible to make useful suggestions
- **BYOK (Bring Your Own Key)** preferred — lets writers control costs and choose models
- **Multiple model support** — different models are better for different tasks
- **Custom prompts** — writers should be able to tune AI behavior to their needs and style
- **AI should work at the scene/beat level** — not try to generate entire plots

## 6. Feature Priority Matrix for Building a Plotting Tool

### Must-Have (Table Stakes)

1. **Visual timeline/swimlane view** — horizontal rows for plotlines, cards for scenes/beats
2. **Drag-and-drop scene reordering** — fluid card movement between positions and plotlines
3. **Color coding** — per storyline/plotline for instant visual differentiation
4. **Scene cards with rich data** — title, synopsis, POV, characters present, notes
5. **Character profiles** — name, bio, traits, images, linked to scenes where they appear
6. **Chapter/act grouping** — nest scenes within chapters, chapters within acts
7. **Story structure templates** — at minimum: 3-Act, Hero's Journey, Save the Cat, Story Circle
8. **Notes system** — freeform notes linked to story elements

### High Value (Differentiators)

9. **Multiple view modes** — timeline, grid/matrix, outline, single-scene
10. **Story bible/codex** — central wiki shared across books in a series
11. **Character arc tracking** — template-based arc milestones on the timeline
12. **Subplot management** — dedicated plotlines for subplots with visual weave
13. **Relationship mapping** — visual web of character relationships
14. **AI brainstorming** — chat-based AI that knows the story context
15. **AI scene beats** — describe what happens, get AI suggestions for execution
16. **Export to writing formats** — Word, Scrivener, plain text

### Nice-to-Have (Power Features)

17. **Fictional calendar** — custom time systems for fantasy/sci-fi
18. **Appearance heatmap** — where characters appear across the story
19. **Characters per scene** — density tracking
20. **Emotional arc graph** — sentiment/tension visualization
21. **Interactive maps** — geographical worldbuilding
22. **Plot hole detection** — AI-powered consistency analysis
23. **Real-time collaboration** — multi-user editing
24. **Gantt chart view** — timeline at different zoom levels
25. **Version history** — explore alternate plotlines while preserving originals

## 7. Key Takeaways for Manuscript's Plot Feature

### What to Learn From Each Tool

| Tool | Key Lesson |
|------|------------|
| **Plottr** | The visual timeline with parallel swimlanes is THE killer feature. Templates that teach story structure are extremely popular. |
| **NovelCrafter** | AI integration done right: context-aware, scene-level, brainstorming-focused. Multiple views of the same data. The Codex concept is powerful. |
| **Campfire** | Modular depth — let writers add complexity as needed. Fictional calendars and Gantt views appeal to fantasy writers. |
| **Scrivener** | Direct connection between outline and manuscript is the ideal — the corkboard IS the manuscript. |
| **Wavemaker** | Multiple planning paradigms (snowflake, kanban, grid, timeline) for different thinking styles. Lightweight and accessible. |
| **Shaxpir** | Automated emotional arc analysis is a unique and compelling feature. |

### What Writers Actually Want

Based on reviews and feature analysis across all tools:

1. **See the whole story at a glance** — visual spatial representation, not text lists
2. **Flexible structure** — support plotters, pantsers, and everything in between
3. **Connected data** — characters linked to scenes, scenes linked to plotlines, everything searchable
4. **Story structure guidance** — templates that help writers learn and apply proven frameworks
5. **Low friction** — fast to add ideas, easy to reorganize, no complex setup required
6. **AI as a thinking partner** — brainstorm, check consistency, suggest possibilities
7. **Manuscript connection** — the outline should flow into or connect with the actual writing
